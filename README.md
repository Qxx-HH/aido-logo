# AIDO GitHub Pages

This folder is ready to publish with GitHub Pages.

Files:

- `index.html`: mobile-friendly landing page for the QR code
- `aido-logo.pdf`: source PDF

Publishing steps:

1. Create a new public GitHub repository.
2. Upload every file in this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main`, folder `/ (root)`, then save.
6. Wait for Pages to finish publishing.

Final URL pattern:

- `https://<your-github-name>.github.io/<repo-name>/`

That URL is the one to encode into the final QR code.
